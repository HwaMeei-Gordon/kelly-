# Kelly Ski Japan ❄️ — 日本滑雪一站式服務網站

給想去日本滑雪的客人一個清楚的入口：介紹服務（課程、教練、住宿、交通）、
熱門雪場、常見問題，並透過「需求單表單」把客人的需求一次整理成一則格式化訊息，
一鍵傳到 Kelly 的 LINE —— 取代目前用 LINE 一來一回逐項詢問的流程。

## 網站內容

- **首頁 Hero**：一句話說明服務，導引客人填需求單
- **服務內容**：滑雪課程／教練、住宿、交通接駁、行程規劃
- **預約流程**：填需求單 → 收到方案報價 → 確認預訂 → 出發
- **熱門雪場**：二世谷、留壽都、富良野、白馬、野澤溫泉、藏王
- **常見問題 FAQ**
- **需求單表單**：人數、日期、程度、板類、需要的服務、雪場、預算、備註
  → 自動產生格式化訊息 → 複製／一鍵開 LINE 傳給 Kelly

純靜態網站（單一 `index.html`，無後端），手機優先設計，客人從 LINE 點開就能直接用。

## 上線前必改：填入 Kelly 的 LINE ID

打開 `index.html`，找到最下方 `<script>` 裡的設定區：

```js
// ====== 設定（Kelly 上線前請修改這裡）======
const KELLY_LINE_ID = "";
```

- **LINE 官方帳號**：填入含 `@` 的 ID，例如 `"@123abcd"`。
  客人按「用 LINE 傳給 Kelly」會直接開啟聊天室並帶入訊息。
- **個人帳號**：填加好友連結最後的 ID，例如 `"kelly123"`
  （即 `line.me/ti/p/~kelly123`）。訊息會自動複製，客人加好友後貼上即可。

> 建議申請 LINE 官方帳號（免費），體驗最順：訊息可以直接預填、不用手動貼上。

## 如何部署（免費，用 GitHub Pages）

1. 進入這個 repo 的 **Settings → Pages**
2. Source 選 **Deploy from a branch**，Branch 選 `main`、資料夾選 `/ (root)`，按 Save
3. 幾分鐘後網站就會出現在 `https://<帳號>.github.io/<repo 名稱>/`
4. 把這個網址放到 Kelly 的 LINE 個人檔案／社群貼文，客人點進來填單即可

## 本機預覽

直接用瀏覽器打開 `index.html` 就能看，不需要安裝任何東西。
